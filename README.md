# TaskTrack Lite

A minimal Android task tracker: add tasks, mark complete, delete, and keep them saved locally.

## Features
- Add a task
- Toggle complete (tap)
- Delete a task
- Persistent storage (TinyDB on App Inventor)

## Tech
- MIT App Inventor (blocks)
- Local storage only (no accounts/APIs)

## How to Run (App Inventor)
1. Open [MIT App Inventor](https://ai2.appinventor.mit.edu).
2. Import the `.aia` from this repo.
3. Connect AI Companion or build APK, then test.

## Screens/Wireframes
- Home: text box + Add button + ListView of tasks
- Tap task to toggle ✓ complete
- Delete button removes selected

## Roadmap
